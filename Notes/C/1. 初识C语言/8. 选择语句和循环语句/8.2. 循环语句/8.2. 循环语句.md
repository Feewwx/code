####
**循环语句:** `while` 语句 `do-while` 语句 `for` 语句

```c
#include <stdio.h>

int main() {
    int line = 0;
    printf("进入大学\n");
    while (line < 20) {
        printf("写代码:%d\n", line);
        line++;
    }
    if (line >= 20) {
        printf("好offer\n");
    } else {
        printf("继续加油\n");
    }
    return 0;
}
```
