####
` !  -  +  &  sizeof  ~  --  ++  * `

`!` - 逻辑反操作

`-` - 负值

`+` - 正值

`&` - 取地址

`sizeof` - 操作数的类型长度(以字节为单位)

`~` - 对一个数的二进制按位取反

`--` - 前置,后置--

`++` - 前置,后置++

`*` - 间接访问操作符(解引用操作符)

**逻辑反操作(!):**

```c
if (flag) {
    // flag为真,进入if
}
if (!flag) {
    // flag为假,进入if
}
```

**对一个数的二进制位按位取反(~):**

```c
#include <stdio.h>

int main() {
    int a = 0;
    printf("%d\n", ~a);  // -1
    return 0;
}
```
`000000000000000000000 - 0的补码`
0=>1,1=>0
`111111111111111111111 - ~0的补码`
|-> 反推反码: 111111111111111111110
|-> 反推原码: 100000000000000000001 - `结果(-1)的原码`
