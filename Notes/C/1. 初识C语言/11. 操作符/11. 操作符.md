**条件操作符:** `exp1(真/假) ? exp2(运行/不运行) : exp3(不运行/运行)`

```c
#include <stdio.h>

int main() {
    int a = 0;
    scanf("%d", &a);
    int b = 10;
    a > b ? printf("a > b\n") : printf("a <= b\n");
    return 0;
}
```

**逗号操作符:** 可以将多个表达式用逗号隔开,并将它们作为一个表达式来运算
> **特点:** ` 从左向右依次计算,整个表达式的结果是最后一个表达式的结果 `

```c
#include <stdio.h>

int main() {
    int a = 10;
    int b = 20;
    int c = 0;
    int d = (c = a - 2, a = b + c, c - 3);
    printf("d = %d\n", d);
    return 0;
}
```
